หมายเหตุสำคัญ

  • LINE API เพิ่มขึ้น: เนื่องจากมี notification หลากหลาย (promotion, เตือนนักเรียน, แจ้งผู้ปกครอง, แจ้งตารางเรียน, ติดตามลูกค้า)

  • ลดค่าใช้จ่าย: ใช้ VPS แทน AWS ลดเหลือ ~1,500-2,200 บาท/เดือน
  • Data Analytics: รายงานการใช้งานและสถิติต่างๆ จะช่วยวิเคราะห์ ROI ของระบบ
  • Security & Compliance: มีระบบ Backup อัตโนมัติทุกวัน และเข้ารหัสข้อมูลตาม PDPA
  • Performance Monitoring: ติดตั้งระบบ monitoring เพื่อแจ้งเตือนปัญหาก่อนผู้ใช้พบ
  • การอัปเดต: Minor updates ฟรี 3 เดือนแรก, Major updates คิดตาม man-hour จริง
